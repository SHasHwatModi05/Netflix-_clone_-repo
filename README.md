# Netflix Clone

A visually accurate clone of the Netflix landing page built using HTML and CSS.  
This project focuses on recreating the modern UI/UX design of Netflix with responsive layouts and clean styling.

---

## 🌐 Live Demo

🔗https://garudaax.netlify.app/

---

## 📌 Overview

This project is a frontend web development exercise aimed at replicating the core design elements of Netflix’s homepage. It demonstrates the use of semantic HTML, structured CSS, and responsive design principles.

---

## ✨ Features

- Pixel-perfect Netflix-inspired UI  
- Fully responsive layout  
- Hero section with background image  
- Modern navigation bar  
- Styled content sections  
- Interactive FAQ section  
- Clean and organized code structure  

---

## 🛠 Technologies Used

- HTML5  
- CSS3  
- Google Fonts  
- GitHub Pages (Deployment)
